# Legal Landing Page — Spanish Law Firm

Conversion-focused landing page built for a personal injury law firm targeting Spanish-speaking clients.

## Features
- Click-to-call hero section
- Lead capture form
- Client testimonials
- Floating WhatsApp button
- Google Tag Manager integration
- Fully responsive (mobile & desktop)

## Tech Stack
HTML, CSS, JavaScript
